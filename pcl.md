## PCL
### 1. Compile and Run Based on [Link](http://www.pointclouds.org/documentation/tutorials/using_pcl_pcl_config.php#using-pcl-pcl-config)
Compile according to the `CMakeLists.txt` file in path `/myproject/path`
```
$ cd /myproject/path
$ mkdir build
$ cd build
$ cmake
```
Build
```
$ make
```
test
```
$ ./generated_test
```
